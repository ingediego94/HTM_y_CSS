Página de enlaces favoritos.

Como la web es tan grande, y cada día crece mas y mas, es indispensable contar con marcadores que nos permitan guardar nuestros sitios web preferidos para cuando no recordamos su url específica.

A continucación presento los cinco enlaces a mis sitios web preferidos.

  1. Netflix. Es una pagina ampliamente conocida para la oferta de contenido streaming como peliculas y series de televisión. Muy uilizada para practicar inglés.
  2. BBC Learning English. Es una página web dedicada a la enseñanza y perfeccionamiento del idioma inglés como segunda lengua.
  3. British Council. Es otro web site similar al anterior, donde su enfoque es enseñar y perfeccionar el inglés como segunda lenga.
  4. Cheatsheets. Es una página que se dedica a recopilar y mostrar los principales comandos que se usan en cada uno de los diferentes lenguages de programación.
  5. DataLemur. Permite practicar SQL mediante consultas en grandes bases de datos, con ejercicios reales de la industria tech.
